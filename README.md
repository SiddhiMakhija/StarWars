# Star Wars Character Generator ⚔️

A simple and interactive React.js application that randomly generates Star Wars characters using the SWAPI (Star Wars API). With a single click, the app fetches and displays character details, combining API integration with dynamic user interaction.

## Overview
This project demonstrates the practical use of React fundamentals and external APIs, focusing on asynchronous data fetching, clean component structure, and responsive UI updates.

## Features
- Random Star Wars character generation  
- Real-time data fetching from SWAPI  
- Asynchronous handling using fetch  
- Dynamic rendering of character details  
- Clean and user-friendly interface  

## Tech Stack
- React.js  
- JavaScript (ES6+)  
- HTML & CSS  
- SWAPI (Star Wars API)  

## How It Works
On user interaction, the application generates a random character ID, requests data from SWAPI, and renders the retrieved information instantly.
